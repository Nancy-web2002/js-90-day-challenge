# 90-Day JavaScript Challenge

A structured, self-paced journey through JavaScript fundamentals, from variables and functions to async APIs, ES6 modules, and modern syntax. This repo documents daily progress with real, working mini-projects (no tutorials copy or pasted, every project was built and debugged from scratch).

**GitHub:** [github.com/Nancy-web2002](https://github.com/Nancy-web2002)

---

##  About This Challenge

I restarted my JavaScript learning journey after realizing I had gaps to close in my fundamentals. This repo is the result, a day-by-day build-up from core syntax to real-world features like live API integration, form validation, and persistent data storage.

Each day includes 2–3 mini-projects, progressively combining new concepts with everything learned before it.

---

##  Technologies Used

- **JavaScript (ES6+)** - variables, functions, arrays, objects, destructuring, spread/rest, template literals, async/await
- **HTML5 & CSS3** - semantic structure, responsive layout, basic styling
- **DOM Manipulation** - dynamic content, event listeners, form handling
- **Fetch API** - live data from external APIs (weather, movies, random users, jokes)
- **localStorage**- persistent client-side data
- **ES6 Modules** - `import`/`export` for organized, multi-file projects
- **Git & GitHub** - version control with conventional commit messages

## How to Run Any Project

1. Clone or download this repository
2. Open the relevant project folder (e.g. `DAY 5/notes-app/`)
3. Open `index.html` directly in your browser
4. For console-only projects (some Day 3 projects), open the browser DevTools console to view output

No build tools or installs required, everything runs with plain HTML/CSS/JS.
